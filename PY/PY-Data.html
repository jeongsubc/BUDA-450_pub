4  Data Types

This section introduces built-in data types in Python.

4.1 Built-in Data Types in Python

Python has various built-in data types, categorized as follows:

Category Data Type Example
Numeric int 20
Numeric float 20.5
Numeric complex 1j
Text str "Hello World"
Sequence list ["apple", "banana", "cherry"]
Sequence tuple ("apple", "banana", "cherry")
Sequence range range(6)
Mapping dict {"name": "John", "age": 36}
Set set {"apple", "banana", "cherry"}
Set frozenset frozenset({"apple", "banana", "cherry"})
Boolean bool True
Binary bytes b"Hello"
Binary bytearray bytearray(5)
Binary memoryview memoryview(bytes(5))
None NoneType None

All expression evaluate to a single value. In the above examples, the expression evaluated to single numerical value. Numerical values come in two basic forms:

4.2 Numbers

There are three numeric types in Python. Let’s enter each type in Python and check it by using the type() that returns the type of the specified object.

  • int: Int, or integer, is a whole number, positive or negative, without decimals, of unlimited length.
# int
print( 123 ) 
print( type( 123 ) )
123
<class 'int'>
  • float: Float, or “floating point number” is a number, positive or negative, containing one or more decimals.
print( 3.14 )
print( type( 3.14 ) )
3.14
<class 'float'>
- Float can also be scientific numbers with an "e" to indicate the power of 10.
print( 987e5 )
print( 1.23e-3 )
98700000.0
0.00123
  • complex: Complex numbers are written with a real number + a j as the imaginary part:
print( 1.5+4j )
print( type( 1.5+4j ) )
(1.5+4j)
<class 'complex'>
int( 3.14 )
3

4.3 Strings

A snippet of text in Python is called a string. Strings in python are surrounded by either single quotation marks ', or double quotation marks ".

For example, 'hello' is the same as "hello". To evaluate whether two values are equal, we can use two equals signs == between them. The expression will evaluate to either True or False (see below for Boolean value).

'hello' == "hello"
True

A string can contain letters, spaces, line breaks, and numbers. When a non-text value is embraced by quotation marks, it becomes a string, and Python treats it differently. For example, an integer 5 is converted to a string with quotation marks.

print( type("5") )
<class 'str'>
# Compare an integer with the same meaning in other types
print( 5 == 5.0 )    # integer vs float
print( 5 == '5' )    # integer vs string 
True
False

The distinction between each of these data types may seem unimportant, but it is. This is because an operator performs different operations, depending on the data type.

For example, the addition + for strings conbine two strings, called concatenation, exactly as they are in the order of the strings in the expression:

# Combine the strings 'Hello' and 'World'
'hello' + ' world'
'hello world'

However, using such operators may cause an error when the operators are applied to different types (e.g., numeric and non-numeric).

# Try adding a string to an integer
'5' + 7

4.4 Boolean Values

In programming, you often need to know whether an expression is true or not. Booleans is a way to represent one of two possible values, True or False.

In Python, Boolean values can be handled by entering exactly True or False (case-sensitive).

print( False )
print( type(False) )
False
<class 'bool'>

In Python, the Boolean values True and False can be converted into numeric values 1 and 0, respectively.

print( int(True) )
print( float(False) )
1
0.0

4.4.1 Comparison Operators

Comparison operators in Python are used to compare two values.

Operator Name Example
== Equal x == y
!= Not equal x != y
> Greater than x > y
< Less than x < y
>= Greater than or equal to x >= y
<= Less than or equal to x <= y

As shown in the example about strings, the == operator returns True if two values are equal to each other.

print( 5 == 5.0 )             # If the values are the same, an integer and a float are the same
print( 5 == "five" )          # A string cannot be equal to a float or an integer.
print( "Five" == "five" )     # Strings with different cases are not the same.
True
False
False

4.4.2 Logical Operators

Logical operators are used to combine Boolean values (e.g., from conditional statements).

Operator Description Example
and True if both statements are true x < 5 and x < 10
or True if at least one statement is true x < 5 or x < 4
not Reverses the result (Boolean negation) not (x < 5 and x < 10)

The operation with a logical operator for two Boolean values returns a single Boolean value.

For example, when both statements are True, the and operator returns True

# When (condition one is True) AND (condition two is True):
cond1 = 1 < 6
cond2 = 4 < 6
print( cond1 and cond2 )
print( True and True )
True
True
# When (condition one is True) AND (condition two is False):
print( True and False )
False

Considering two possible values of the statements, all possible combinations and their and results can be organized in a Truth table:

Condition 1 Logical operator Condition 2 Evaluation
True and True True
True and False False
False and True False
False and False False

Similarly, create a Truth table that shows all possible results from or operator with two statements.

Condition 1 Logical operator Condition 2 Evaluation
True and True True
True and False True
False and True True
False and False False

Lastly, the not operator only operates on a single expression for Boolean negation (i.e., flipping True to False or False to `True.

# The not operator flips a True to False
print(not True)
print(not False)
False
True

4.5 Lists

Lists are one of four built-in data types that are used to store multiple items. Lists are created using square brackets [], and each item of any data type can be included with delimiter , in such ways of [item1, item2, item3, item4, ...].

# A list of three integers 
int_list = [2, 4, 6, 8]
print( int_list )
print( type( int_list ) )
[2, 4, 6, 8]
<class 'list'>
# A list of three mixed types 
mixed_list = [-0.2, 'car', False]
print( mixed_list )
print( type( mixed_list ) )
[-0.2, 'car', False]
<class 'list'>
# An empty list
empty_list = []
print( empty_list )
print( type( empty_list ) )
[]
<class 'list'>
# A list of lists
lists_list = [ int_list, mixed_list, empty_list] 
print( lists_list )
print( type( lists_list ) )
[[2, 4, 6, 8], [-0.2, 'car', False], []]
<class 'list'>

List items are presented in the order stored, and can be referred to by using the index.

print(int_list[0])     # first item of int_list
print(int_list[-1])     # the last item of int_list
2
8

We can change the value of any item in a list using an assignment statement that contains the item’s index number.

# Changing the value of an item in a list
Animal = ['Cat', 'Dog', 'Bear']
print(Animal)
['Cat', 'Dog', 'Bear']
# Changing an item using an assignment state with index number
Animal[1] = 'Lion'
print(Animal)
['Cat', 'Lion', 'Bear']

4.6 Dictionaries

Dictionaries are another built-in data types storing multiple items. Dictionaries are created using curly brackets {}, and each item has a unique key paired with some value by : such that { key1 : value1, key2 : value2, key3 : value3, ...}

# A dictionary of three integers 
int_dict = {"Math": 95, "History": 55, "Art": 70}
print( int_dict )
print( type( int_dict ) )
{'Math': 95, 'History': 55, 'Art': 70}
<class 'dict'>
# A dictionary of mixed values
mixed_dict = {"Age": 23, "State": "WV", "Membership": True, 'Ordered': ["Shoes", "Socks"]}
print( mixed_dict )
print( type( mixed_dict ) )
{'Age': 23, 'State': 'WV', 'Membership': True, 'Ordered': ['Shoes', 'Socks']}
<class 'dict'>

Dictionary items are presented in key:value pairs, and can be referred to by using the key name.

print(mixed_dict["Ordered"])
['Shoes', 'Socks']

To access a item of a list in the dictionary, an additional bracket with its index can be placed after the first indexing expression.

print(mixed_dict["Ordered"][1])
Socks
