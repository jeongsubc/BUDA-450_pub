7  Data Inspection

7.1 Overview

Often, data cannot be used directly for your analysis. For accurate, meaningful outcomes, data have to be prepared, prior to the application of data mining tools.

“This material outlines basic methods for preparing data in R, which is essential for analysis. It covers inspecting data, organizing data by subsetting and sorting, integrating multiple datasets, cleaning data with missing elements, and transforming data by applying mathematical operations, binning, and combining categories.”

7.2 Preliminaries

7.2.1 Packages

7.2.2 Data

Let’s start by creating a DataFrame object for three attributes X1, X2, X3 using pandas.DataFrame():

df = pd.DataFrame(
  {
    "X1": [1, 2, 3, 5, 4, 3],
    "X2": ["Neutral", "Satisfied", "Dissatisfied", "Dissatisfied", "Satisfied", "Satisfied"],
    "X3": [0.5, None, 0.2, 0.1, 0.9, np.nan]
  }
)

7.3.1.1 Renaming Rows/Columns

In pandas, row indices are typically numeric by default. You can assign custom labels using dat.index:

df.index = [f"rec{i+1}" for i in range(len(df))]
df
X1 X2 X3
rec1 1 Neutral 0.5
rec2 2 Satisfied NaN
rec3 3 Dissatisfied 0.2
rec4 5 Dissatisfied 0.1

To store original index values in a new column before reindexing:

df_reset = df.copy()
df_reset = df_reset.reset_index() # '.reset_index(drop=True)' to not keep the old index

7.3.2 Accessing/Modifying Metadata

While pandas limits attributes to its defined structure (e.g., df.shape), you can attach arbitrary metadata3 using the .attrs dictionary:

df.attrs
{}
df.attrs["comment"] = "This dataset is for a class example in BUDA-450."
df.attrs["custom_id"] = [12, 23, 34, 45, 54]
df.attrs
{'comment': 'This dataset is for a class example in BUDA-450.',
 'custom_id': [12, 23, 34, 45, 54]}

This metadata will not be displayed in print(dat) or dat.info(); it’s purely for internal use.

7.4 Organizing Data

7.4.1 Subsetting Data

Intro to data structure Indexing and Selecting Data

Often, you may need only a part of large data. With pandas, subsetting data is easy and powerful.

7.4.1.1 Element Selection

You can select a single element using iloc (by integer index) or loc (by label):

df.loc["rec3", "X2"]   # by row label and column name
#df.loc["rec3", 1]      # row name and column index
'Dissatisfied'
df.iloc[2, 1]          # 3rd row, 2nd column (indexing starts at 0)
df.iloc[2]["X2"]       # chaining style
'Dissatisfied'

Note: You must first set custom row names to use labels like “rec3”:

df.loc["rec3"]
X1               3
X2    Dissatisfied
X3             0.2
Name: rec3, dtype: object
df.index = [f"rec{i+1}" for i in range(len(df))]

7.4.1.2 Entire Column/Row Selection

To select an entire column, we can include:

df.loc[:,"X2"]       # 2nd column by index
rec1         Neutral
rec2       Satisfied
rec3    Dissatisfied
rec4    Dissatisfied
rec5       Satisfied
rec6       Satisfied
Name: X2, dtype: object
df.iloc[:, 1]       # 2nd column by index
rec1         Neutral
rec2       Satisfied
rec3    Dissatisfied
rec4    Dissatisfied
rec5       Satisfied
rec6       Satisfied
Name: X2, dtype: object

Alternatively,

df["X2"]            # by column name
df.X2               # attribute-style access (if no space or special char in column name)
rec1         Neutral
rec2       Satisfied
rec3    Dissatisfied
rec4    Dissatisfied
rec5       Satisfied
rec6       Satisfied
Name: X2, dtype: object

To select an entire row:

df.iloc[1, :]       # 2nd row by index
df.loc["rec2", :]   # by row name
X1            2
X2    Satisfied
X3          NaN
Name: rec2, dtype: object

7.4.1.3 Multiple Selection

To select multiple columns/rows, a list of labels or indices can be used

df.iloc[:, [0, 2]]         # 1st and 3rd columns
df.iloc[[1, 4], :]         # 2nd and 5th rows
#df.iloc[[1, 4], [0, 2]]    # intersection of those rows and columns
X1 X2 X3
rec2 2 Satisfied NaN
rec5 4 Satisfied 0.9 
df.loc[:, ['X1', 'X3']]    # 1st and 3rd columns
df.loc[['rec2', 'rec5'], :]         # 2nd and 5th rows
#dat.iloc[[1, 4], [0, 2]]    # intersection of those rows and columns
X1 X2 X3
rec2 2 Satisfied NaN
rec5 4 Satisfied 0.9

7.4.1.4 Conditional Selection

You can use a logical condition, which shows whether a condition for each element is met, to filter rows.

For example, df["X1"] > 3 returns a Boolean Series of logical test results for all the elements of the Series df["X1"] :

condition = df["X1"] > 3
print(condition)
rec1    False
rec2    False
rec3    False
rec4     True
rec5     True
rec6    False
Name: X1, dtype: bool

Use this Boolean mask to filter rows:

# with saved condition
df[condition]  # or directly 'df[df["X1"] > 3]'
X1 X2 X3
rec4 5 Dissatisfied 0.1
rec5 4 Satisfied 0.9

For the logical operation, review the materials in BP-Logics.

7.4.1.5 Assigning New Values to Selection

You can modify selected data by assigning new values, where the shape of the selected object (i.e., the left-hand side of the statement) is the same with the size of the shape of the new values (i.e., the right-hand side).

df.iloc[[1], [0, 2]] = [-100, -200] # overwrite the 1st and 3rd element of 2nd row with [-100, -200]
print(df)

rec1    1       Neutral    0.5
rec2 -100     Satisfied -200.0
rec3    3  Dissatisfied    0.2
rec4    5  Dissatisfied    0.1
rec5    4     Satisfied    0.9
rec6    3     Satisfied    NaN

7.4.2 Sorting Data

Data can be examined by simply checking the extreme (e.g., largest or smallest) values after sorting the values.

7.4.2.1 Sorting DataFrame

To sort rows in a DataFrame based on values of a column, use .sort_values():

df.sort_values(by="X1")                                   # by a Single Column in Ascending)

rec3 3 Dissatisfied 0.2
rec6 3 Satisfied NaN
rec5 4 Satisfied 0.9
rec4 5 Dissatisfied 0.1 
df.sort_values(by=["X1", "X3"])                           # by Multiple Columns (Ascending)
X1 X2 X3
rec2 -100 Satisfied -200.0
rec1 1 Neutral 0.5
rec3 3 Dissatisfied 0.2
rec6 3 Satisfied NaN
rec5 4 Satisfied 0.9
rec4 5 Dissatisfied 0.1
