8  Pandas

This section introduces Pandas, a powerful Python package for data manipulation and analysis, providing flexible data structures and tools for handling structured datasets.

Pandas is a Python package that provides powerful and flexible data structures for data analysis and manipulation. Particularly, two main structures, Series and DataFrame, in Pandas are designed to make working with structured data intuitive and efficient.

  • Series is a one-dimensional labeled array capable of holding data of any type (integers, strings, Python objects, etc.)
  • DataFrame is a two-dimensional labeled array (i.e., matrix) with columns of potentially different types.

8.1 Series

A Series object consists of a one-dimensional array (i.e., vector) of values, whose elements can be referenced using an index array.

8.1.1 Creating Series

You can create a Series object from a list, a numpy array, or a Python dictionary.

For example, the following shows how to create a Series object with a list.

import numpy as np   # for later usages
import pandas as pd 

# creating a series from a list
list_data = [3.1, 2.9, -1.7, -0.2, 2.9, 4.5]
sr = pd.Series(list_data)  
print('sr =\n', sr, '\n', sep="")

print('sr.values =', sr.values)                   # display values of the Series
print('sr.index  =', sr.index)                     # display indices of the Series
print('sr.dtype  =', sr.dtype)                     # display the element type of the Series

A Series object can be created with a numpy object assigning indexes together. For example, when a data array np.arange(3,7) is given with indexes ['a', 'b', 'c', 'd']:

# creating a series from a numpy ndarray
arr_data = np.arange(3,7)
arr_index = ['a', 'b', 'c', 'd']
sr_arr = pd.Series(arr_data, index=arr_index) 
print('sr_arr =\n', sr_arr, '\n', sep="")
sr_arr =
a    3
b    4
c    5
d    6
dtype: int64

Another way is with a dictionary—the key becomes the index while the the value of the key is taken as the corresponding element’s value. For example, a Series object can be defined from a dictionary {'MI': 'Lansing', 'CA': 'Sacramento', 'TX': 'Austin', 'MN': 'St Paul'}:

# creating a series from a dictionary 
dict_data = {'MI': 'Lansing', 'CA': 'Sacramento', 'TX': 'Austin', 'MN': 'St Paul'}
sr_dict = pd.Series(dict_data)                
print('sr_dict =\n', sr_dict, '\n', sep="")
sr_dict =

CA    Sacramento

MN       St Paul

8.1.2 Indexing and Slicing

Two typical indexing methods for the value of an element in a Series are based on either the label or the integer index of element(s):

  • using .loc[] with a group of rows and columns by labels or boolean array (https://pandas.pydata.org/docs/reference/api/pandas.DataFrame.loc.html)
  • using .iloc[] with a group of rows and columns by integer indexes (https://pandas.pydata.org/docs/reference/api/pandas.DataFrame.iloc.html)
# Access elements based on labels or a boolean array
print(sr_labeled.loc['Rec3'])
print(sr_labeled.loc[sr_labeled == -0.2])

8.2 DataFrame

A DataFrame object is a tabular, spreadsheet-like data structure containing a collection of columns, each of which can be of different types (numeric, string, boolean, etc). Unlike Series, a DataFrame has distinct row and column indices.

There are many ways to create a DataFrame object (e.g., from a dictionary, list of tuples, or even numpy’s ndarrays).

# creating a DataFrame from a dictionary
cars = {
    'make': ['Ford', 'Honda', 'Toyota', 'Tesla'],
    'model': ['Taurus', 'Accord', 'Camry', 'Model S'],
    'MSRP': [27595, 23570, 23495, 68000],
}
df_car = pd.DataFrame(cars, index=['A','B','C','D'])

print('df_car =\n', df_car, "\n", sep="")

print('df_car.values =\n', df_car.values, "\n", sep="")  # the values 
print('df_car.shape =\n', df_car.shape, "\n", sep="")    # the shape (# rows, # cols)
print('df_car.size  =\n', df_car.size, "\n", sep="")     # the number of all elements

print('df_car.index   =\n', df_car.index, "\n", sep="")  # the row indices
print('df_car.columns =\n', df_car.columns)              # the column labels
