Appendix A — Markdown Cheat Sheet

This notebook is based on Markdown cheat sheet to provide a quick overview of the Markdown syntax elements. The elements introduced in this notebook are supported by many other Markdown applications. For more detailed, comprehensive information, refer to the reference guides from basic syntax and extended syntax or Tutorials.

A.1 Keyboard Shortcuts for Cell Types in Jupyter

  • y Change to Code Cell Type
  • m Change to Markdown Cell Type and
  • r Change to Raw Cell Type
